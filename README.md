# 💰 Expense Tracker

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Flask](https://img.shields.io/badge/Flask-Web%20Framework-black)
![Database](https://img.shields.io/badge/Database-SQLite-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

A simple **Expense Tracker web application** built using **Python and Flask**.  
It allows users to record, update, and manage their daily expenses efficiently.

---

## 🚀 Features

- ➕ Add new expenses
- ✏️ Edit existing expenses
- ❌ Delete expenses
- 📊 View all expenses in one place
- 📁 Export expenses to CSV
- 💾 SQLite database storage
- 🎨 Simple and responsive UI

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Backend logic |
| Flask | Web framework |
| SQLite | Database |
| HTML | Frontend |

## ⚙️ Installation

### 1️⃣ Clone the repository

```

git clone [https://github.com/Pavithra329/expense_tracker.git](https://github.com/Pavithra329/expense_tracker.git)

```

### 2️⃣ Navigate to the project folder

```

cd expense_tracker

```

### 3️⃣ Create a virtual environment

```

python -m venv venv

```

### 4️⃣ Activate the environment

**Windows**

```

venv\Scripts\activate

```

**Mac/Linux**

```

source venv/bin/activate

```

### 5️⃣ Install dependencies

```

pip install -r requirements.txt

```

---

## ▶️ Run the Application

```

python app.py

```

Open your browser and visit:

```

[http://127.0.0.1:5000/](http://127.0.0.1:5000/)

```
